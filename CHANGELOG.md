# 更新日志

## 0.1.21

- 修复切换显示归档后树视图会话行缩进视觉不稳定的问题，改用更稳定的缩进字符。
- 将“切换归档”命令文案改为“显示/隐藏归档会话”，避免与单个会话的归档操作混淆。

## 0.1.20

- 移除 `onStartupFinished` 激活事件，避免 VS Code 启动阶段过早触发官方 Codex Webview。
- 保留打开会话管理视图和执行命令时的按需激活，降低与官方 Codex 插件 Webview 初始化的耦合。

## 0.1.19

- 新增会话快照 JSON 磁盘缓存，存放在 VS Code 扩展 `globalStorage`，首次进入视图会先展示上次缓存，再后台刷新真实数据并覆盖缓存。
- 搜索并打开会话优先使用当前内存快照或磁盘缓存，避免每次搜索都重新全量拉取会话记录。
- 本工作区、全部工作区、归档显示和旧过滤命令改为基于当前快照即时重算，减少无意义的全量刷新。
- 新增 `codexSessionManager.enableDiskCache` 配置项，默认开启，可按需关闭缓存。
- 收敛侧边栏标题栏搜索入口，只保留“搜索并打开会话”，避免两个搜索入口混淆。

## 0.1.18

- 修复首次点击其他工作区会话时“正在切换”通知阻塞实际切换的问题，点击后会立即切换工作区。
- 工作区切换后会在扩展启动时自动恢复待打开会话，并对官方 Codex 会话打开增加短重试，减少需要重复点击的问题。
- 新增“搜索并打开会话”入口，可按标题、预览、备注、路径、项目、分支和会话 ID 搜索所有工作区与归档会话。
- 搜索过滤改为插件本地统一过滤，避免 app-server 预过滤导致本地别名、备注、项目标签无法搜到。

## 0.1.17

- 将扩展 manifest 的显示名和描述改为字面中英文双语值，避免扩展详情页或 Marketplace 缓存回落到旧英文标题。
- 将 VS Code 扩展详情页 README 改为中英文双语介绍，覆盖功能、要求、说明和打包步骤。
- 同步根 README 的中英文项目介绍、功能说明和发布说明。

## 0.1.16

- 将扩展显示名改为 `Codex 会话管理 / Codex Session Manager`，方便中文和英文环境识别。
- 将 Marketplace 与 VS Code 扩展描述改为中英文双语，明确这是按项目管理 Codex 会话并可跳转官方 Codex 继续对话的扩展。

## 0.1.15

- 新增 Marketplace 扩展 PNG 图标，避免网页上传时缩略图或图标解析为空。
- 重新生成用于 Marketplace 上传的 VSIX 包。

## 0.1.14

- 将扩展发布者切换为 `wangzhilin777`，准备发布到 VS Code Marketplace。
- 补充 Marketplace 所需的仓库、主页、问题反馈和关键词元数据。
- 更新 README 与 LICENSE 中的发布信息。

## 0.1.13

- 修复 app-server 数据源下本地重命名、置顶、未读、备注等右键操作刷新后不生效的问题。
- 非复制类右键操作补充明确反馈，避免点击后看起来“没反应”。
- 打开工作区、资源管理器定位等路径类操作在路径缺失时会提示原因。

## 0.1.12

- 在归档会话右键菜单中新增“删除已归档会话”，并要求二次确认。
- “取消归档”新增二次确认，行为对齐桌面端归档查询中的操作入口。
- 删除操作只允许处理 `CODEX_HOME/sessions` 或 `CODEX_HOME/archived_sessions` 下的会话 JSONL 文件，避免误删工作区文件。
- 参考桌面端右键菜单，新增本地置顶、标记未读、复制工作目录、复制深度链接、在资源管理器中打开、在新窗口打开工作区、派生到本地终端等适合 VS Code 的操作。
- 置顶与未读为插件本地状态：置顶会影响插件列表排序，未读会在会话描述中显示标识。

## 0.1.11

- 优化会话行操作区分：官方 Codex 入口改为带火箭图标的主内联操作。
- “打开本地详情页”移入右键打开分组，并增加书本图标，避免与官方 Codex 入口并排显示时混淆。
- 更新中英文命令标题与悬浮提示，明确单击会话默认在官方 Codex 继续。

## 0.1.10

- 修复刚创建或运行中的会话在 `session_index.jsonl` 尚未写入标题时，列表只显示 session ID 的问题。
- 新增 rollout JSONL 标题回退解析：索引无标题时，从真实用户消息中提取首行作为会话标题。
- 自动跳过 `AGENTS.md`、环境上下文、图片 wrapper 等系统性文本，避免误把注入上下文当成标题。

## 0.1.9

- 新增按工作区/项目聚合 Codex 会话，并支持所有工作区与本工作区切换。
- 支持点击会话跳转官方 Codex 继续对话。
- 归档会话归入对应项目下的“已归档”子文件夹，并保留醒目的归档标识和图标。
- 点击归档会话时自动取消归档；CLI 不可用时使用本地文件回迁兜底。
- 修复官方 Codex 重命名后仍显示旧标题的问题。
- 修复短时间内 VS Code 同步产生重复会话的问题。
- 修复缺失索引时间的会话排序沉底问题，使用 rollout 文件时间兜底。
- 支持 VS Code 中英文语言环境。
- 增加会话加载统计日志，方便排查数据源和过滤问题。
