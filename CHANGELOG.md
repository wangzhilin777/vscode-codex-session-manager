# 更新日志

## 0.1.10

- 修复刚创建或运行中的会话在 `session_index.jsonl` 尚未写入标题时，列表只显示 session ID 的问题。
- 新增 rollout JSONL 标题回退解析：索引无标题时，从真实用户消息中提取首行作为会话标题。
- 自动跳过 `AGENTS.md`、环境上下文、图片 wrapper 等系统性文本，避免误把注入上下文当成标题。

## 0.1.9

- 新增按工作区/项目聚合 Codex 会话，并支持所有工作区与本工作区切换。
- 支持点击会话跳转官方 Codex 继续对话。
- 归档会话归入对应项目下的“已归档”子文件夹，并保留醒目的归档标识和图标。
- 点击归档会话时自动取消归档；CLI 不可用时使用本地文件回迁兜底。
- 修复官方 Codex 重命名后仍显示旧标题的问题。
- 修复短时间内 VS Code 同步产生重复会话的问题。
- 修复缺失索引时间的会话排序沉底问题，使用 rollout 文件时间兜底。
- 支持 VS Code 中英文语言环境。
- 增加会话加载统计日志，方便排查数据源和过滤问题。
